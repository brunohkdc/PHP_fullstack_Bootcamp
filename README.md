# PHP_fullstack_Bootcamp
Bootcamp for Fullstack development in PHP

<h2>Intstructions</h2>
Run composer install to install all packages needed for the app.<br />
Copy .env.example to .env, edit .env to your settings. Ensure database settings are correct.<br />
Run php artisan key:generate to generate a new app key.<br />
Run npm install to install Node packages.<br />
Run php artisan migrate to create tables in DB.<br />
Run npm run dev and php artisan serve to start dev server.<br />
